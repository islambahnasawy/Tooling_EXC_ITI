Description: 
	    This batch scripts allows you to create multiple folders of projects 
	    with 3 files: TEXT.txt, SOURCE.c, Header.h inside each folder.
	    You can set the names of folders and files you want to generate by 
	    modifying the input.txt file in /TXT folder.
	    But make sure to write the names as follows: 
	    		
	    		folder1 text1.txt source1.source header1.h
	    		folder2 text2.txt source2.source header2.h
	    		folder3 text3.txt source3.source header3.h
	    
	    where: folderX is the folder 	name,
	    	textX 	is the text 	file name,
	    	sourceX is the source 	file name,
	    	headerX is the header 	file name.
	    	
/**********************************************************/




User Manual:
	1. Modify the input.txt file in /TXT folder with the names you want.	
	2. RUN only the run.bat file.
